INTO THE DREAMLANDS

Into the dreamlands é um jogo couch coop split screen de plataformas e puzle inspirado em jogos com It takes two. Este jogo retrata a história de uma menina que está a passar pela fase onde começa a deixa a sua imaginação de criança para passar a ser uma adolescente. Cada um dos jogadores controla uma personagem sendo elas a menina e outra o seu amigo imaginário. A mecânica principal deste jogo é o facto das duas personagens verem o ambiente de formas diferentes. Esta mecânica é suportada pela imaginação da criança, que vai diminuindo ao longo do jogo.

DODGE GAME

Dodge game é um jogo inspirado em jogos como Super Hexagone e Shapes and Beats.
Neste jogo o jogador pode escolher entre dois modos de jogo.
Levels, 8 niveis que vão aumentando a dificuldade onde o jogador tem de aguentar 30 segundos sem ser atingido por qualquer objeto.
Survive, modo endless onde o jogador vai somar pontos quanto mais tempo aguentar em jogo.
Este jogo tem como mecânicas um dash e o reflect, que permite ao jogador refletir um bola que destroi qualquer objeto que tocará

BABEL

Babel é um jogo first person arena type onde podemos atirar uma adaga e teletransportar para ela a qualquer momento. Neste jogo o objetivo é progredir na torre de babel passando pelos diferentes pisos que tem diferentes inimigos. Neste jogo o jogador perde vida overtime tendo que matar os inimigos para recuperar vida. Para além disso pode ser necessário a ativação de pressure plates para completar o nivel. Este jogo conta com um boss.

DISTURBING SITE

Disturbing site é um jogo onde o nosso único objetivo é chatear os trabalhadores de uma obra, roubando materias para que a obra não possa avançar. Mas para complicar, existem trabalhadores que vão vigiando a área tendo que nos esconder deles enquanto roubamos os materiais, aumentado o tempo que a obra vai necessitar para ficar completa. Para além disso existe a mecânica de colocar uma trap no chão para que os trabalhadores fiquem parados por uns segundos.
